# 📚 Atividade de HTML, CSS e JavaScript

Este projeto reúne uma série de exercícios práticos com o objetivo de aprender e aplicar conceitos fundamentais de desenvolvimento web utilizando **HTML**, **CSS** e **JavaScript**.

---

## 🚀 Conteúdos abordados

### 🧱 Estrutura HTML Semântica
- Uso de tags como:
  - `header`
  - `nav`
  - `main`
  - `section`
  - `aside`
  - `footer`
- Organização correta do conteúdo da página

---

### 🎨 CSS e Box Model
- Aplicação de:
  - `margin` (espaço externo)
  - `padding` (espaço interno)
  - `border` (borda)
- Estilização de elementos
- Diferença entre margin e padding
- Criação de layouts simples

---

### 📦 Flexbox
- Alinhamento de elementos
- Distribuição de espaço
- Propriedades utilizadas:
  - `display: flex`
  - `justify-content`
  - `align-items`
  - `flex-direction`
  - `gap`

---

### 🧩 Componentes criados
- Página estruturada com layout completo
- Caixa demonstrando o Box Model
- Comparação entre margin e padding
- Card informativo
- Cards alinhados com Flexbox
- Card interativo com botão

---

### ⚙️ JavaScript (Interatividade)
- Manipulação de elementos com:
  - `getElementById`
- Uso de eventos:
  - `onclick`
- Alteração dinâmica de estilos (mudança de cor)

---

## 🧪 Exercícios realizados

1. Estrutura HTML semântica
2. Estilização com Box Model
3. Comparação entre margin e padding
4. Aplicação prática em layout completo
5. Criação de card informativo
6. Layout com Flexbox (linha e coluna)
7. Interatividade com botão
8. Card interativo (desafio)

---

## 🎯 Objetivo

O objetivo desta atividade é desenvolver uma base sólida em:
- Estruturação de páginas web
- Estilização com CSS
- Organização de layout com Flexbox
- Introdução à interatividade com JavaScript
